HTML5, CSS3 and jQuery

Header Text

HTML5

We will learn HTML5 semantic tags, multimedia with HTML5, how to create HTML5 forms, and the HTML5 APIs that use JavaScript. The APIs include Canvas, Drag and Drop, and Web Storage.

CSS3

CSS3 introduces new concepts such as border-radius and box-shadow. HTML5 and CSS3 introduce ways to replace images or JavaScript.

jQuery

jQuery is a JavaScript DOM manipulation library. It is supported in any browser that supports JavaScript. The jQuery UI library and the jQuery mobile library use the jQuery library.

In Conclusion

This website puts together HTML5 and CSS3 for Responsive Web Design (RWD).
